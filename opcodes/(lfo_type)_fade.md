lang: en
title: (lfo type)_fade
---
LFO fade-in effect time, in seconds.

##### Examples

```
amplfo_fade=1
pitchfo_fade=0.4
```

Useful for vibrato which grows in intensity gradually,
as commonly performed by bowed string players.

| Type  | Default | Range      |
| ---   | ---     | ---        |
| float | 0       | 0 to 100 s |
