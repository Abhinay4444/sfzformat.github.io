Filter velocity tracking - the amount by which the cutoff changes
with incoming note velocity. Measured in cents.

##### Examples

```
fil_veltrack=0

fil_veltrack=1200
```

| Type    | Default | Range               |
| ---     | ---     | ---                 |
| integer | 0       | -9600 to 9600 cents |
