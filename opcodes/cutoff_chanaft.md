# cutoff_chanaft

The variation in the cutoff frequency when
MIDI channel aftertouch messages are received, in cents.

##### Examples

```
cutoff_chanaft=1200

cutoff_chanaft=-100
```

| Type    | Default | Range               |
| ---     | ---     | ---                 |
| integer | 0       | -9600 to 9600 cents |
