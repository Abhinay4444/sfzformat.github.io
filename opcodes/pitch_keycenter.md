Root key for the [sample](sample).

For samples which only need to be played at their
natural pitch and triggered by one specific MIDI note, it's generally easier to
use [key](key) instead.

##### Examples

```
pitch_keycenter=56

pitch_keycenter=c#2
```

| Type    | Default      | Range     |
| ---     | ---          | ---       |
| integer | 60 (C4)      | 0 to 127  |
|         |              | C-1 to G9 |
