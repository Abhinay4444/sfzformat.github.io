# eqN_freq

Frequency of the equalizer band, in Hertz.

##### Examples

```
eq1_freq=80
eq2_freq=1000
eq3_freq=4500
```

| Type  | Default       | Range         |
| ---   | ---           | ---           |
| float | eq1_freq=50   | 0 to 30000 Hz |
|       | eq2_freq=500  |               |
|       | eq3_freq=5000 |               |

| Modulation Sources
|           ---
| Envelope | ✓ |
| LFO      | ✓ |
| MIDI CC  | ✓ | eqN_freq_onccX
