# pan_veltrack

The effect of note velocity on panning. With positive values, higher-velocity
notes will be panned more to the right, with negative values more to the left.

##### Examples

```
pan_veltrack=100
pan_veltrack=-40
```

| Type  | Default | Range         |
| ---   | ---     | ---           |
| float | 0.0     | -100 to 100 % |
