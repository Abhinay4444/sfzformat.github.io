# bend_step

Pitch bend step, in cents.

##### Examples

```
bend_step=100 // glissando in semitones

bend_step=200 // glissando in whole tones

bend_step=1200 //leap an entire octave at a time
```

| Type    | Default | Range     |
| ---     | ---     | ---       |
| integer | 1       | 1 to 1200 |
