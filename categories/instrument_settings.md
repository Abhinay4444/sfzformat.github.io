| ---                                     | ---     |

| [octave_offset](/opcodes/octave_offset) | SFZ v2  |
| [set_ccN](/opcodes/set_ccN)             | SFZ v2  |
| [#define](/directives/define)           | SFZ v2  |
| [label_ccN](/opcodes/label_ccN)         |  TODO   |
| [set_hdccN](/opcodes/set_hdccN)         |  ARIA   |
